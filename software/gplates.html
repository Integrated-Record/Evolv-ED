GPlates

Website: https://www.gplates.org/
License: GPL-2
Source: https://github.com/GPlates/GPlates

Description

GPlates is a plate tectonics program. Manipulate reconstructions of geological and paleogeographic features through geological time. Interactively visualize vector, raster and volume data. PyGPlates is the GPlates Python library.
